# BERT Fine-Tuning for Sentiment Analysis

## Project Overview
This project demonstrates fine-tuning of a pre-trained BERT model on the IMDB dataset for sentiment classification.

## Technologies Used
- Python
- PyTorch
- Hugging Face Transformers
- Scikit-learn

## Workflow
Raw Data → Preprocessing → Tokenization → Model Training → Evaluation

## Results
- Accuracy: 85%
- Precision: 86%
- Recall: 81%
- F1 Score: 83%
